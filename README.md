
This repository includes the source code of "**Heterophilous to Homophilous Graph Transformation Based on Deep Generative Framework**"

# Requirements

* Pytorch 
* python 3.x
* networkx
* scikit-learn
* scipy
* dgl

# How to run
* run `python3 train.py --dataset minesweeper --model HeToHo --num_layers 1`
